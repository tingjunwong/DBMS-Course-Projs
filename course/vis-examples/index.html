A few simple d3.js examples

To try these, you should clone the github repo and fire up a web server, e.g.: 


  % python -m SimpleHTTPServer 8888

  1. styling text with d3 awkwardly, and with method chaining
  2. data-DOM joins: data-driven text scaling
  3. outer joins: using enter() and exit() sets
  4. bar charts using simple HTML divs and SVG rects
  5. data-driven circles
  6. theta-join: binding data with key functions and with simple text labels added
